## 硬件环境
| 硬件模块       | 型号/参数                          | 作用                     |
|----------------|------------------------------------|--------------------------|
| 主控芯片       | STM32F103C8T6                      | 核心控制、OTA 逻辑处理   |
| 存储芯片       | W25Q64（8MB SPI Flash）| 存储待升级固件镜像       |
| 存储芯片       | 24C02（2KB I2C EEPROM）| 存储升级状态、版本信息   |
| 通信模块       | USART（可扩展蓝牙/WiFi）| 接收固件数据             |

flowchart TD
    A[初始化] --> B[读取24C02升级状态]
    B --> C{是否有升级指令}
    C -- 否 --> D[正常运行当前固件]
    C -- 是 --> E[擦除W25Q64固件区]
    E --> F[接收固件数据并写入Flash]
    F --> G[CRC32校验固件]
    G -- 失败 --> H[回滚至原固件+更新状态]
    G -- 成功 --> I[更新24C02升级状态]
    I --> J[重启并加载新固件]
